# LinuxGuard Security Suite

LinuxGuard is an advanced Linux security monitoring and intrusion detection toolkit developed using Bash scripting on Kali Linux.

## Features

- File Permission Auditing
- Intrusion Detection System (IDS)
- Network Monitoring
- Process Monitoring
- Real-Time System Monitoring
- IP Blocking System
- Security Report Generation
- Firewall Rule Automation
- Professional CLI Dashboard

## Technologies Used

- Bash Scripting
- Linux System Utilities
- iptables
- netstat
- ss
- awk
- grep

## Installation

```bash
chmod +x install.sh
sudo ./install.sh
```

## Running LinuxGuard

```bash
sudo ./main.sh
```
